# 🐞 Bug Tracker System

A simple and beginner-friendly **Bug Tracker Application** developed using **Java**.  
This project helps to manage and track software bugs efficiently.

---

## ✨ Features

✅ Report New Bugs  
✅ View All Bugs  
✅ Update Bug Status  
✅ Delete Bugs  
✅ Simple Console-Based Menu  
✅ Easy to Understand Code Structure  

---

## 🛠️ Technologies Used

- ☕ Java
- 🌱 Git
- 🐙 GitHub

---

## 📂 Project Modules

| Module | Description |
|---|---|
| Dashboard | Shows bug summary |
| Report Bug | Add new bug details |
| View Bugs | Display all bugs |
| Update Bug | Change bug status |
| Delete Bug | Remove bug records |

---

## 🔄 Bug Status Flow

```text
Open → In Progress → Resolved → Closed
```
